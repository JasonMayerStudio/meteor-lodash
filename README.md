A wrapper for [Lo-Dash](http://lodash.com) v3.1.0, a JavaScript utility library delivering consistency, modularity, performance, & extras. Lo-Dash can be used as a drop-in replacement for [Underscore.js](http://underscorejs.org).

#### Usage
```
meteor add erasaur:meteor-lodash
```

The package exports `_` and `lodash` as two global variables.

#### Why use Lo-Dash?
Lo-Dash is known to be generally more performant than Underscore, and has a set of [features](https://lodash.com/#features) that Underscore lacks.
